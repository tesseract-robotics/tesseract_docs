Find tool center point provided in the manipulator info. 

If manipulator information tcp is defined as a string it does the following
\begin{DoxyItemize}
\item First check if manipulator info is empty or already an Isometry3d, if so return identity
\item Next if not, it checks if the tcp offset name is a link in the environment if so throw an exception.
\item Next if not found, it looks up the tcp name in the SRDF kinematics information
\item Next if not found, it leverages the user defined callbacks to try an locate the tcp information.
\item Next throw an exception, because no tcp information was located.
\end{DoxyItemize}
